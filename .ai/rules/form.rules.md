# Form Rules

## Library
- Use react-hook-form
- Use zod validation

## Structure
- Label + input + error
- Use FormField component

## UX
- Disable submit when loading
- Show error message
- Show success toast

## Performance
- Use controlled inputs only when needed
- Use `useDebounce` hook (from `src/hooks/useDebounce.ts`) for search inputs.

## Accessibility
- label + htmlFor
- name attribute required
